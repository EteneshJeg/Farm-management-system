Farm Dashboard Project

This project is a farm management dashboard built using Vite, React, JavaScript, Tailwind CSS, and shadcn-ui. It provides a clean interface and a fast development environment suitable for dashboard and data-driven applications.

Technologies Used

Vite
React
JavaScript
Tailwind CSS
shadcn-ui

Getting Started

Make sure you have Node.js and npm installed.

1. Clone the repository
   git clone https://github.com/EteneshJeg/Farm-management-system

2. Navigate into the project folder
   cd Farm-management-system

3. Install dependencies
   npm install

4. Start the development server
   npm run dev

The application will be available at the local development URL shown in the terminal.
